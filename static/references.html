References for Vancomycin Dosing

Key guidance and formulas implemented in this application:

  • AUC/MIC target: Maintain daily AUC between 400–600 mg·h/L (assuming MIC=1 mg/L) per the 2020 ASHP/IDSA/PIDS/SIDP consensus guideline. Exposures above 800 mg·h/L increase nephrotoxicity risk.
  • When AUC monitoring is unavailable, trough concentrations of 15–20 mg/L can be used as a surrogate to achieve AUC/MIC ≥400.
  • Dosing based on actual body weight: Loading 20–25 mg/kg for serious infections; maintenance 15–20 mg/kg. Round doses to the nearest 250 mg. Avoid total daily doses ≥100 mg/kg.
  • Interval by creatinine clearance (CrCl): q8h if CrCl ≥100 mL/min, q12h if 60–100 mL/min, q24h if <60 mL/min.
  • Pharmacokinetic approximations: ke ≈ 0.00083 × CrCl + 0.0044 (h⁻¹); Vd ≈ 0.7 L/kg × weight. Use first-order elimination to compute half-life and AUC.
